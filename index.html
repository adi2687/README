Our Development Roadmap

We're building the future of the internet, one phase at a time.

Phase 1: Core Infrastructure

  • Host HTML/CSS/JS on IPFS
  • Link to ENS/Handshake (decentralized DNS)
  • Enable access via Brave or custom browser/extension

Phase 2: Identity & Messaging

  • Integrate wallet login (MetaMask, SpruceID)
  • Add optional messaging (XMTP/libp2p)

Phase 3: Decentralized Search

  • Build crawler to index IPFS pages
  • Store metadata in IPFS/DHT
  • Build minimal search frontend

Phase 4: Incentives & Contribution

  • Build publishing portal for creators
  • Add token rewards via smart contracts

Phase 5: UI/UX Wrapper

  • Build dashboard/portal to view web content
  • Enable content upload and management
  • Enhance search and discovery features
